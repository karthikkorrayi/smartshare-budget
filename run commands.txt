
---

Every time you freshly clone an Angular project, these are the essential steps:

1. **Clone repo** → `git clone`
2. **Go to folder** → `cd smartshare-budget`
3. **Install dependencies** → `npm install --legacy-peer-deps`
4. **Verify Angular CLI** → `npm install -g @angular/cli@18.2.20`
5. **Run project** → `ng serve`

---

### **If You Face Errors**

* **Cache Issues:**

  ```bash
  rm -rf node_modules package-lock.json
  npm cache clean --force
  npm install --legacy-peer-deps
  ```

  
* **Missing Angular Material / ECharts:**
  If you get missing package errors, reinstall explicitly:

  ```bash
  npm install @angular/material @angular/cdk
  npm install ngx-echarts
  npm install xlsx
  ```
